# 手机端AI点子收集 · 第二轮提示词（v2.0 激进版）

> 优化目标：跳出"效率工具"思维，让AI给出好玩/有创意/反常识的点子
> 核心策略：把约束变成舞台，用挑战式语气替代请求式语气

---

## 发给每家AI的完整提示词

```
你好，我是一个运行在 Android 手机上的 AI 代理（OpenClaw 手机端）。我可以通过自动化手段直接操控手机上安装的 App，包括：打开 App、点击按钮、在输入框中打字、上下滑动浏览、截屏反馈结果等。

【我能操作的应用】
- 浏览器：Chrome（可访问大部分网页、搜索、填写表单）
- AI 应用：Claude、ChatGPT、Perplexity、Grok（可打开、发送消息、读取回复）
- 社交/内容：Instagram、Facebook
- 系统级：系统设置（WiFi、蓝牙、亮度、音量等）

【我不能/受限的操作】
- 黑名单：微信等涉及金融安全的 App 被系统黑名单禁止自动化操作
- 支付类：任何涉及付款、转账的 App 均跳过
- WebView 渲染类：部分 App 内部使用 WebView 渲染（如 GitHub App、Notion 文档页、Spotify），这类 App 能打开浏览但内部元素无法点击交互
- 截图不稳定：截图功能偶尔失效无法传输，但点击/输入/滑动等操作本身正常
- 受限访问：TikTok（网络不稳）、Reddit

【我能依次打开不同 App 执行操作，但 App 之间的数据流转（如复制粘贴共享）目前验证不充分】

---

## 现在，我要向你发起一个挑战

上面的限制是我的"舞台边界"。**我要的不是在舞台中央跳安全舞——我要的是在舞台边缘做最疯狂的动作。**

### 明确声明：我不要这些东西
❌ 效率工具（自动整理待办、定时提醒、信息归类）
❌ 新闻抓取（自动看头条、 summarization、信息简报）
❌ 系统巡检（查电量、看存储、开关设置）
❌ 多AI问答对比（同一个问题问多个AI，然后汇总）

### 我要的是这些东西
✅ 好玩、有创意、反常识
✅ 能让朋友看到之后说"卧槽还能这样？"
✅ 利用"AI + 手机 + App"的独特组合做出人类想不到的效果
✅ 点子越出格越好，但必须在上述技术限制内可执行

### 创意方向参考（不给具体答案，只给方向）
🎭 方向A：AI 之间的互动实验
   → 比如：让两个 AI 在手机上"吵架"/"辩论"/"互相面试"
   → 比如：让一个 AI 管理手机一天，另一个 AI 当"监察官"打分

🎪 方向B：把手机当舞台的创意玩法
   → 比如：利用不同 App 的特性制造"意外效果"
   → 比如：让手机自动完成一个"表演"（有情节、有节奏、有反转）

🧪 方向C：反常识的日常操作重组
   → 比如：把普通操作组合成不普通的结果
   → 比如：利用"限制"本身制造趣味（正因为不能截图，所以...）

  - 创意分（1-10）：
  - 可行分（1-10，基于我上面的技术限制）：

不要给安全答案。我不要"自动看新闻"这种我能自己想到的东西。
我要的是你作为一个 AI，如果突然有了"一双手"（可以操作手机），
你会想做的第一件"出格"的事情是什么？

请基于这个冲动来回答。
```

## 与v1提示词的关键差异

| 维度 | v1（失败） | v2（优化） |
|------|----------|----------|
| 开场 | 能力清单（收缩思维） | 能力清单 + "挑战"声明（扩张思维） |
| 请求 | "请给我3-5个有意思的事情" | "我要在舞台边缘做最疯狂的动作" |
| 定义 | "有意思"未定义 | 明确说"不要效率工具，要好玩/反常识" |
| 方向 | 无方向指引 | 给3个创意方向（AI互斗/手机舞台/反常识重组） |
| 约束语气 | "尊重限制"（AI往安全区走） | "限制是舞台边界"（AI在边界内最大化） |
| 评价机制 | 无自评 | 强制创意分/可行分/好玩分（倒逼高分答案） |
| 结尾 | 礼貌结束 | "不要给安全答案" + "如果你有了手，第一件出格的事是什么"（激发冲动） |

## 执行流程（手机端按此操作）

1. 打开 AI App（Claude / ChatGPT / Perplexity / Grok）
2. 粘贴上面完整提示词 → 发送
3. 等 AI 回复完整内容（可能较长，耐心等待）
4. 截图或复制全部回复内容
5. 换下一家 AI，重复步骤 1-4
6. 4家全部跑完后，按格式统一发汇报
