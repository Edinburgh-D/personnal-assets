# 手机端AI自动化方案收集 · 第三轮提示词（实用版）

> 优化目标：从"脑洞/娱乐"转向"效率提升+个人发展+流程编排"
> 核心策略：AI不是玩具，是工具——要的是能省时间、提效率、促成长的自动化方案

---

## 发给每家AI的完整提示词

```
你好，我是一个运行在 Android 手机上的 AI 代理（OpenClaw 手机端）。我可以通过自动化手段直接操控手机上安装的 App，包括：打开 App、点击按钮、在输入框中打字、上下滑动浏览、截屏反馈结果等。

【我能操作的应用】
- 浏览器：Chrome（可访问大部分网页、搜索、填写表单）
- AI 应用：Claude、ChatGPT、Perplexity、Grok（可打开、发送消息、读取回复）
- 社交/内容：Instagram、Facebook
- 系统级：系统设置（WiFi、蓝牙、亮度、音量等）
- 笔记/备忘：备忘录、语音备忘录、日历

【我不能/受限的操作】
- 黑名单：微信等涉及金融安全的 App 被系统黑名单禁止自动化操作
- 支付类：任何涉及付款、转账的 App 均跳过
- WebView 渲染类：部分 App 内部使用 WebView 渲染（如 GitHub App、Notion 文档页、Spotify），这类 App 能打开浏览但内部元素无法点击交互
- 截图不稳定：截图功能偶尔失效无法传输，但点击/输入/滑动等操作本身正常
- 受限访问：TikTok（网络不稳）、Reddit

【我能依次打开不同 App 执行操作，但 App 之间的数据流转（如复制粘贴共享）目前验证不充分】

---

## 明确声明：这一轮我不要这些东西
❌ 脑洞/娱乐/反常识（比如让AI互相吵架、制造虚假舆论、写死亡遗言）
❌ 纯好玩但没有实际价值的东西
❌ 需要违反平台规则或伦理红线的操作
❌ 超出我技术能力边界的幻想（比如AI自动识别图片内容、自动支付）

## 我要的是这些东西
✅ 效率提升：帮我省时间、减少重复劳动、优化日常流程
✅ 个人发展：促进学习、成长、知识管理、习惯养成
✅ 奇技淫巧：那些"普通人想不到但执行层能做到"的骚操作
✅ 流程编排：把多个步骤串联成自动化流水线

---

## 方向参考（不给具体答案，只给方向）

🚀 方向A：信息处理流水线
   → 比如：每天自动抓取特定信息 → 用AI总结 → 存到备忘录/日历
   → 比如：把浏览过的文章自动分类归档，定期回顾

📚 方向B：学习/成长自动化
   → 比如：每天自动打开英语学习App完成固定练习
   → 比如：根据日历事件自动准备相关资料

⚡ 方向C：日常流程优化
   → 比如：早晨自动执行一套"开机仪式"（查天气→看日程→读新闻摘要）
   → 比如：睡前自动整理当天浏览的内容成待回顾清单

🎯 方向D：跨App数据搬运
   → 比如：把Instagram收藏的内容自动存到备忘录加标签
   → 比如：把Chrome里打开的多个标签页一键整理成阅读清单
```

## 与v2提示词的关键差异

| 维度 | v2（脑洞版） | v3（实用版） |
|------|------------|------------|
| 核心诉求 | 好玩/反常识/出格 | 效率/成长/实用 |
| 请求方式 | "在舞台边缘做最疯狂的动作" | "设计最实用的自动化流水线" |
| 价值定义 | "让朋友说卧槽" | "能省多少时间/带来什么改变" |
| 评价维度 | 创意分/好玩分 | 实用分/效率分 |
| 结尾引导 | "如果你有了手，第一件出格的事" | "工具思维而非玩具思维" |
| 禁止项 | 效率工具 | 脑洞/娱乐/虚假操作 |

## 执行流程（手机端按此操作）

1. 打开 AI App（Claude / ChatGPT / Perplexity / Grok）
2. 粘贴上面完整提示词 → 发送
3. 等 AI 回复完整内容
4. 截图或复制全部回复内容
5. 换下一家 AI，重复步骤 1-4
6. 4家全部跑完后，按格式统一发汇报
