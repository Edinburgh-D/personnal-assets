## 设计原则

1. **无标准答案**：每题4个选项，映射真实育儿风格
2. **场景真实**：全部来自家长高频冲突场景
3. **选项等长**：25-35字，保持阅读节奏一致
4. **理论支撑**：每个选项标注心理学理论来源
