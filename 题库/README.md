## 📊 题库结构

- **4个年龄段**：0-3岁 / 4-6岁 / 7-12岁 / 13-18岁
- **每年龄段6道题**，共24道
- **每道题4个选项**：A/B/C/D 对应四种育儿风格

| 选项 | 风格 | 画像 |
|------|------|------|
| A | 情绪回应型 | 温室园丁 🟢 |
| B | 规则锚定型 | 目标教练 🟡 |
| C | 探索追问型 | 侦探型父母 🔵 |
| D | 行动果断型 | 行动派指挥官 🟤 |

**计分**：选A/B/C/D各得对应画像1分，最高分为主画像，同分=混合型。
