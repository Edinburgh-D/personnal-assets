# 育儿预演所 · 6道测评题完整版

（以下6题全部由"育儿预演所"情报素材提炼，每题4个选项无标准答案，做完后生成育儿风格画像）

---

## 第1题：超市玩具区的"地面作战"

**场景**：你推着购物车，3岁半的儿子在玩具区突然停下，指着货架上一辆118元的遥控车大喊"我要这个！"你说"家里已经有5辆车了，今天不买玩具。"他愣了两秒，随即把购物车里的牛奶盒摔到地上，一屁股坐下去开始嚎哭，腿还在地上蹬。周围已经有两位顾客转头看你。

**A.** 蹲下来与他平视，手指轻轻按住他肩膀："你想要那辆车，特别想要，现在没买到你很生气。（停顿）但今天的规矩是不买玩具，这个规矩不会因为大哭就改变。你可以在这儿哭一会儿，我陪着你。"
→ **风格标签**：情绪回应型 + 规则锚定型
→ **理念**：P.E.T.积极倾听 + 正面管教"和善而坚定"
→ **题目解析**：先命名情绪再守住边界，孩子学到的是"我的感受被看见，但规则不因为我的崩溃而动摇"。

**B.** 弯腰凑近他，声音压低："这样，你起来不哭，妈妈现在不买了，但如果你这周每天都能自己收玩具，周末我带你来买这辆——我们拉钩。"
→ **风格标签**：条件协商型 + 延迟满足型
→ **理念**：操作性条件反射（斯金纳）+ 目标激励
→ **题目解析**：把冲突转化为可量化的契约，孩子学到的是"努力可以交换结果"，但需警惕所有互动变成交易。

**C.** 自己也蹲下来，语调平静："你告诉我，为什么非要这辆车？它跟家里那辆哪里不一样？"等他抽抽搭搭说出理由后，再回应。
→ **风格标签**：探索追问型 + 深度沟通型
→ **理念**：儿童中心主义（罗杰斯）+ 需求洞察
→ **题目解析**：不急于解决表面冲突，先理解内在动机。适合高敏感或表达力强的孩子，但可能拉长冲突周期。

**D.** 不说话，直接弯腰把他抱起来（他挣扎着蹬腿），推车往收银台走，离开玩具区后把他放下，等他安静下来才开口。
→ **风格标签**：行动果断型 + 场景隔离型
→ **理念**：行为切断法（森田/脱敏）+ 环境重置
→ **题目解析**：认为激烈情绪中讲道理无效，先物理脱离再复盘。效率高，但需确保孩子事后有机会表达情绪。

## 第2题：写作业拖拉到深夜

**场景**：四年级，周二晚上10点15分，孩子作业还剩语文阅读和数学练习册两科。他已经打了4次哈欠，眼睛红红的，但手里还拿着铅笔在草稿纸上画小人。你提醒了3次"快点写"，他每次都说"知道了"，然后继续画。你明天7点要起床开会。

**A.** 走过去合上他的作业本："今天先睡，明早我6点叫你起来写。身体比作业重要，这个优先级我说了算。"
→ **风格标签**：弹性调整型 + 身体优先型
→ **理念**：发展适宜性实践（DAP）+ 生理节律保护
→ **题目解析**：把"健康"放在"完成任务"前面，传递的是"你这个人比你的作业更重要"。但需避免养成"拖到最后有人兜底"的预期。

**B.** 坐在旁边，手指敲了敲计划表："我们约好了10点收摊，现在过了15分钟。没写完的部分你自己跟老师解释，我不会帮你写请假条。"
→ **风格标签**：契约维护型 + 自然后果型
→ **理念**：正面管教"逻辑后果"+ 责任自担
→ **题目解析**：让孩子体验"约定被打破"的真实后果。关键是之前确实有共同制定的规则，而非临时起意的惩罚。

**C.** 拉过一把椅子坐下："是哪一科卡住了？是阅读题看不懂，还是数学那道应用题？我们一起把它拆开，做完这一科就睡。"
→ **风格标签**：根因排查型 + 脚手架型
→ **理念**：维果茨基"最近发展区"+ 共同建构
→ **题目解析**：不把"拖拉"定义为态度问题，而是当作"任务太难"或"启动困难"来排查。适合学业卡点明确的孩子。

**D.** 什么都不说，关灯回自己房间。第二天早上他困得起不来，你也没叫他，让他自己面对迟到和没交作业。
→ **风格标签**：完全放权型 + 极端自然后果型
→ **理念**：放任主义（极端版）+ "吃一堑长一智"
→ **题目解析**：把选择权完全交给孩子，用真实世界的反馈来教育。风险在于：年龄小的孩子可能无法把"迟到"和"昨晚拖拉"建立因果联系。

## 第3题：孩子撒谎"是小猫干的"

**场景**：你听到客厅"砰"的一声，跑过去看到花瓶碎在地上，水洒了一地。4岁的女儿站在旁边，手指绞着衣角。你问"怎么回事？"她说："小猫从桌子上跳下来碰掉的。"你家的猫上周就送回老家了。

**A.** 蹲下来，声音放柔："花瓶碎了妈妈有点心疼，但你告诉我是小猫干的……你是怕妈妈生气吗？怕妈妈不爱你了吗？"
→ **风格标签**：情绪优先型 + 安全感补给型
→ **理念**：依恋理论（Bowlby）+ 情绪命名
→ **题目解析**：先处理"为什么撒谎"背后的恐惧，再处理撒谎本身。孩子学到的是"说实话不会失去爱"。

**B.** 带她回到现场，指着碎片："我们一起看看，如果是小猫碰的，地上应该有猫爪印或者猫毛。现在我们找到了什么？"
→ **风格标签**：事实重建型 + 认知校准型
→ **理念**：皮亚杰认知发展 + 实证思维训练
→ **题目解析**：用"破案"的方式教孩子区分想象与现实，不直接贴"说谎"标签。适合逻辑思维萌芽期的孩子。

**C.** 表情严肃："不管是谁打碎的，现在我们需要一起打扫。但妈妈要告诉你——说谎比打碎花瓶更严重，因为说谎会让妈妈不知道还能不能相信你。"
→ **风格标签**：规则建立型 + 信任契约型
→ **理念**：科尔伯格道德发展（规则导向）+ 关系信用
→ **题目解析**：明确区分"行为"和"说谎"的严重等级，把重点放在"信任"这个核心关系资产上。

**D.** 故意瞪大眼睛："哇，小猫现在会跳这么高了吗？那下次我们教它用爪子开门吧！"然后递给她扫帚："来，我们一起教小猫怎么打扫战场。"
→ **风格标签**：幽默化解型 + 去羞耻化型
→ **理念**：游戏力（Cohen）+ 情绪脱敏
→ **题目解析**：用幽默消解紧张，让孩子从"防御模式"切换到"合作模式"。适合关系已经有些紧张、需要先破冰的场景。

## 第4题：青春期锁门/摔手机

**场景**：周六下午，14岁的儿子把自己关在房间3小时了。你敲门喊他吃饭，他在里面吼"别烦我！"你把耳朵贴门上，听到他在里面摔了一个东西（后来知道是手机）。你再敲门，他大喊"我说了别烦我！滚！"

**A.** 没再敲门，放了一份饭和一杯水在他门口，贴了一张便签："饭在这儿，饿了拿进去吃。你想聊的时候我都在，不想聊也行。"
→ **风格标签**：空间尊重型 + 安全基地型
→ **理念**：依恋理论（安全基地）+ 青春期自主权
→ **题目解析**：给情绪留出物理和心理空间，传递"我不离开你，但我也不入侵你"。关键是事后要主动开启对话，不能无限期冷处理。

**B.** 站在门外，声音平静但坚定："我知道你现在想独处，但摔东西和喊'滚'是家里的红线。你出来，我们谈5分钟，谈完你继续回去待着。"
→ **风格标签**：温和坚守型 + 底线维护型
→ **理念**：正面管教"尊重与坚定并行"+ 家庭契约
→ **题目解析**：尊重情绪但不纵容行为，明确区分"感受可以接纳"和"行为有边界"。需要家长自身情绪稳定。

**C.** 给他发微信（而不是敲门）："你喊'别烦我'的时候，听起来像遇到了很难很难的事。我不进去，但如果你想说话，我随时都在。如果不想说，也没关系。"
→ **风格标签**：情感翻译型 + 非侵入型
→ **理念**：情绪聚焦疗法（EFT）+ 数字化沟通适配
→ **题目解析**：用文字替代面对面压力，帮他"翻译"自己的情绪（愤怒背后可能是挫败/羞耻/无助）。适合言语对抗激烈的青春期。

**D.** 等到周日下午他平静了，坐下来："昨天摔手机+喊'滚'，这两件事我们需要谈谈后果。手机修好了还你，但下周一到周五没有手机。下次你可以提前说'我需要一小时'，而不是摔东西。"
→ **风格标签**：冷静契约型 + 后果教育型
→ **理念**：逻辑后果 + 替代行为训练
→ **题目解析**：不卷入情绪风暴，等平静后复盘。重点是给出"下次可以怎么做"的具体替代方案，而非单纯惩罚。

## 第5题：二孩家庭老大争宠

**场景**：老二出生3个月了。5岁的老大最近开始尿床（已经半年没尿过了）、故意抢老二的咬胶、在你喂老二奶的时候扯你衣服喊"我也要喝"。奶奶说"你是哥哥了，要让着弟弟"，他听完把弟弟的袜子扔进了垃圾桶。

**A.** 每天固定晚饭后30分钟，爸爸带老二，你带老大去小区散步，只做他想做的事。告诉他："这30分钟只属于你，老二不会来打扰。"
→ **风格标签**：专属时光型 + 情感补给型
→ **理念**：依恋修复 + 质量时间（Quality Time）
→ **题目解析**：用"专属感"对抗"被替代感"，满足的是"我还是被优先爱着的"这个核心需求。关键是真的要做到不被打断。

**B.** 在换尿布的时候喊他："哥哥，只有你能帮妈妈——递一下那个湿巾，你拿得到弟弟够不到的地方。"
→ **风格标签**：角色赋权型 + 价值感重建型
→ **理念**：家庭系统理论 + 角色升级
→ **题目解析**：把"哥哥"身份从"竞争者"重新定义为"被需要的合作者"。但需确保不是每次都让他"帮忙"，否则变成另一种负担。

**C.** 坐下来，一手拉着他一手拉着弟弟："我们来定个公平规则——谁先拿到玩具谁先玩5分钟，然后响铃交换。你觉得5分钟够吗？"
→ **风格标签**：公平裁判型 + 程序正义型
→ **理念**：社会交换理论 + 冲突调解
→ **题目解析**：用"规则"替代"谁大谁让"，把家长从"偏心裁判"变成"规则维护者"。适合能听懂规则的孩子。

**D.** 把他抱到腿上（老二在旁边），看着他的眼睛："你最近经常尿床、扔弟弟袜子，是不是心里有个害怕——怕妈妈爱弟弟不爱你了？"
→ **风格标签**：情感翻译型 + 深层命名型
→ **理念**：情绪聚焦（EFT）+ 内在小孩对话
→ **题目解析**：直接说出孩子说不出口的恐惧，把行为翻译成情感需求。需要家长有足够的情绪敏感度，且时机要准（孩子愿意靠近的时候）。

## 第6题：高敏感孩子因衣服材质崩溃

**场景**：幼儿园通知下周一拍集体照，要求穿红色毛衣。你买了件红色羊毛衫，6岁女儿摸了一下突然尖叫"像针在扎我！"缩到沙发角落发抖，死活不肯试穿。奶奶在旁边说"别矫情了，就穿5分钟拍个照，别的小孩都行怎么就你不行"。

**A.** 给老师发消息："我们穿红色纯棉打底衫代替，颜色一样。我女儿对羊毛材质有触觉敏感，这是生理差异不是矫情。"
→ **风格标签**：感官优先型 + 差异承认型
→ **理念**：感觉统合理论 + 神经多样性（Neurodiversity）
→ **题目解析**：把高敏感当作"需要被适配的特质"而非"需要被纠正的问题"。孩子学到的是"我的身体感受是被尊重的"。

**B.** 把羊毛衫剪一小块下来，让她每天摸5秒，摸完给一个贴纸，集满7个贴纸可以换一次冰淇淋。
→ **风格标签**：渐进脱敏型 + 行为塑造型
→ **理念**：系统脱敏法（Wolpe）+ 代币制
→ **题目解析**：用渐进暴露+正向强化扩展舒适区。适合"确实需要适应某种材质"的场景（如校服），但需注意孩子是否已有过度焦虑。

**C.** 蹲下来："拍集体照要穿红色，但扎人的感觉确实很难受。你能想个办法吗？比如里面穿件不扎的打底，或者我们找找有没有不扎人的红衣服？"
→ **风格标签**：目标协商型 + 问题解决型
→ **理念**：合作问题解决（Ross Greene）+ 自主权赋予
→ **题目解析**：不把"拍集体照"和"舒服"对立，而是邀请孩子参与寻找双赢方案。培养的是"遇到困难可以想办法"的思维模式。

**D.** 把羊毛衫收起来，说"那就不拍了"。第二天跟老师请假："我们身体不舒服，集体照不参加了。"
→ **风格标签**：自主权绝对型 + 边界守护型
→ **理念**：身体主权无条件 + 退出权
→ **题目解析**：传递"你的身体你做主，哪怕代价是不参加集体活动"。激进但明确。风险在于孩子可能学会"不舒服=放弃"，需要家长额外补充"有时候我们可以试试"的弹性教育。

# 育儿风格画像结果页

## 4大主画像

**🟢 温室园丁（A型主导）**
> *"你治愈的不是一个问题，是一个人。"*

你倾向于先处理情绪、再处理事情。孩子在你这里学到的第一课是"我的感受是被接纳的"，这是安全感最牢固的地基。你的挑战在于：规则会不会因为"太想呵护"而变得弹性过大？孩子需要知道，爱是无限的，但边界是清晰的。

**🟡 目标教练（B型主导）**
> *"我不是在管你，我是在帮你长成你自己都认得出的人。"*

你擅长把混沌的冲突转化为可操作的契约和目标。孩子学到的是"努力可以交换结果"，这个世界是有因果的。你的挑战在于：当所有互动都变成交易，孩子会不会忘记"有些事本身值得做，不是因为奖励"？

**🔵 侦探型父母（C型主导）**
> *"我没有在解决问题，我在翻译一门只有我能听懂的语言。"*

你不急于给答案，而是先蹲下来理解"为什么"。孩子在你这里体验到的是"我被真正看见了，而不只是被管理了"。你的挑战在于：过度分析会不会让简单的冲突变得复杂？有时候孩子需要的不是被理解，而是被带领。

**🟤 行动派指挥官（D型主导）**
> *"我给的不是保护，是摔下去之后自己爬起来的证据。"*

你决策果断、执行迅速，孩子适应规则的速度很快。你传递的是"世界是讲规矩的，但你有能力应对"。你的挑战在于：情绪被"高效处理"之后，有没有被真正消化？孩子需要知道，你可以很强，但也可以脆弱。

## 5大混合型

**⚪ 矛盾型（A+D混合）**
> *"你想共情，又想控制；想温柔，又想立威。孩子困惑的不是规则，是'妈妈今天站在哪边'。"*
→ 成长建议：选一个你更自然的方向深耕，另一部分交给另一位家长或外部支持。

**🟣 策略型（B+C混合）**
> *"你既有目标意识，又有洞察能力。但有时候，你过度设计了每一次互动，忘了育儿不是项目管理。"*
→ 成长建议：允许自己"搞砸"，孩子不需要完美父母，需要真实父母。

**🌿 滋养型（A+C混合）**
> *"你是情绪海绵+需求侦探，孩子在你这里几乎不会感到孤独。但你要小心——你不是孩子的情绪容器，你也会满。"*
→ 成长建议：建立自己的"情绪排放系统"， parenting is a marathon, not a sprint.

**⚡ 效率型（B+D混合）**
> *"你相信目标，也相信行动。家庭像一台精密运转的机器——但机器里没有'被浪费的情绪'，只有'被跳过的感受'。"*
→ 成长建议：每周设定一个"无效率时段"，什么都不解决，只是待在一起。

**🌀 摇摆型（C+D混合）**
> *"你想理解，又想控制；想探索，又想决断。你在'懂孩子'和'管孩子'之间来回拉扯。"*
→ 成长建议：区分"哪些事需要理解"（情感冲突）和"哪些事需要决断"（安全/健康红线）。
