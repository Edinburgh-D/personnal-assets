# 「育儿预演所」用户场景预演方案

> 让4家AI（Claude/ChatGPT/Perplexity/Grok）扮演不同类型的用户，模拟使用小程序的完整流程，提出专业修改建议。

---

## 预演目标

不是让AI"评价产品"，而是让AI**扮演真实用户**，走完完整使用流程后说"哪里卡住了、哪里困惑、哪里想放弃"。

---

## 用户角色设计（4个典型画像）

| 角色 | 特征 | 核心诉求 | 可能的行为路径 |
|------|------|---------|-------------|
| **角色A：焦虑新手妈妈** | 孩子2岁，第一次当妈，每天刷小红书看育儿经，容易被吓到 | 求确认"我这样做对吗" | 快速做完测试→反复看结果→想分享求认同 |
| **角色B：理性学习型爸爸** | 孩子5岁，喜欢看书和研究，相信科学育儿 | 求深度解释"为什么这个答案更好" | 慢慢看每道题→仔细阅读解析→想要更多延伸阅读 |
| **角色C：佛系二胎家长** | 老大8岁老二3岁，育儿经验丰富但偶尔松懈 | 求省力"有没有简单有效的办法" | 随便做做→觉得题目太长→想跳过直接看建议 |
| **角色D：冲突型伴侣** | 和配偶育儿理念不一致，经常吵架 | 求证据"你看AI都说我是对的" | 拉配偶一起做→比较两人结果→想生成对比报告 |

---

## 预演流程（每家AI都要走完整路径）

### Step 1：进入小程序
- 看到信息收集页（婚姻状态、是否有孩子、孩子年龄）
- 每个角色根据自己的情况选择

### Step 2：开始测评
- 走完6道场景题
- 每个角色根据自己的性格选择不同答案

### Step 3：查看结果
- 获得育儿风格画像
- 查看详细解析

### Step 4：探索后续
- 看到"今日微行动"推荐
- 看到"分享给伴侣"按钮
- 看到"再测一次"选项

### Step 5：退出/分享
- 决定是否分享到朋友圈
- 决定是否保存结果

## 发给每家AI的提示词（通用版）

```
请你扮演一位真实的用户，体验一个名为"育儿预演所"的微信小程序。

## 你的角色设定
[根据角色A/B/C/D填写]

## 小程序信息
- 产品：育儿预演所（ parenting rehearsal mini-program）
- 功能：通过6道育儿场景选择题，生成用户的"育儿风格画像"，并给出每日微行动建议
- 页面结构：
  1. 信息收集页（婚姻状态、是否有孩子、孩子年龄）
  2. 测评页（6道场景题，每题4个选项）
  3. 结果页（育儿风格画像+详细解析+微行动推荐+分享按钮）

## 请你完成以下任务

**第一步：模拟进入**
假设你打开了这个小程序，看到了信息收集页。
- 你会怎么填写？为什么？
- 这个页面的设计有没有让你困惑的地方？

**第二步：模拟答题**
假设你看到了6道育儿场景题，每题4个选项。
- 你作为这个角色，大概会选什么类型的答案？
- 答题过程中有没有让你想放弃的地方？

**第三步：模拟看结果**
假设你获得了"育儿风格画像"（比如"温室园丁型"、"灯塔型"等）。
- 这个结果描述有没有让你认同或反感？
- 你希望得到什么样的结果？

**第四步：模拟后续操作**
- 你会点击"分享给伴侣"吗？为什么？
- 你会尝试"今日微行动"吗？为什么？
- 你会"再测一次"吗？为什么？

**第五步：提出修改建议**
基于以上模拟体验，请从以下维度提出3-5条具体的修改建议：
1. **文案问题**：哪些文案让你困惑、反感或觉得不真实？
2. **流程问题**：哪个步骤最容易流失用户？为什么？
3. **功能缺失**：你希望有什么但小程序没有的功能？
4. **情感触点**：哪个环节最能打动你？哪个最冷漠？
5. **商业化问题**：如果未来要收费/变现，你最可能为什么付费？

## 输出格式

**角色代入宣言**（1句话，说明你是谁、你现在的状态）

**体验日记**（按Step 1-4，写你的内心OS和操作决策）

**修改建议**（按上面5个维度，3-5条，每条要有具体位置和理由）

**情绪曲线**（1-10分，标注在以下节点：
- 进入信息收集页：__/10
- 开始答题：__/10
- 看到结果：__/10
- 看到微行动：__/10
- 决定是否分享：__/10
)

**最后一句**：如果让你用一句话向朋友推荐或吐槽这个小程序，你会说什么？
```

## 4家AI的分工（建议）

| AI | 扮演角色 | 原因 |
|----|---------|------|
| Claude | 角色B（理性学习型爸爸） | Claude擅长结构化分析，能给深度建议 |
| ChatGPT | 角色A（焦虑新手妈妈） | ChatGPT擅长共情，能模拟焦虑情绪 |
| Perplexity | 角色D（冲突型伴侣） | Perplexity擅长信息整合，能给出数据支撑的建议 |
| Grok | 角色C（佛系二胎家长） | Grok风格活泼，能模拟"随便做做"的佛系心态 |

## 手机端执行流程

1. 打开 Claude → 粘贴"角色B版"提示词 → 等待完整回复 → 截图/复制
2. 打开 ChatGPT → 粘贴"角色A版"提示词 → 等待完整回复 → 截图/复制
3. 打开 Perplexity → 粘贴"角色D版"提示词 → 等待完整回复 → 截图/复制

5. 统一发汇报（包含4家AI的完整回复）
