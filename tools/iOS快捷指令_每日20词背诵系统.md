# iOS 快捷指令：每日20词背诵系统

**创建日期**：2026-05-07  
**难度**：⭐⭐（跟着步骤做即可）  
**耗时**：首次 setup 约15分钟

---

## 方案概览

```
iCloud Drive/words.txt  (单词库)
         ↓
    快捷指令读取
         ↓
   随机抽取20行
         ↓
  输出到备忘录+通知
```

**优势**：无需API费用、离线可用、可自定义词库、支持艾宾浩斯复习

---

## 第一步：准备单词库文件

### 1.1 在电脑或手机上创建文件

在 **iCloud Drive** 根目录（或任意文件夹）创建一个纯文本文件：

**文件名**：`vocabulary.txt`

**文件格式**（每行一个单词，用 `|` 分隔字段）：

```
abandon|/əˈbændən/|v. 放弃，抛弃
ability|/əˈbɪləti/|n. 能力，才能
aboard|/əˈbɔːrd/|adv. 在船上，在飞机上
absent|/ˈæbsənt/|adj. 缺席的，不在场的
absolute|/ˈæbsəluːt/|adj. 绝对的，完全的
absorb|/əbˈzɔːrb/|v. 吸收，吸引
abstract|/ˈæbstrækt/|adj. 抽象的
abundant|/əˈbʌndənt/|adj. 丰富的，充裕的
abuse|/əˈbjuːs/|n./v. 滥用，虐待
academic|/ˌækəˈdemɪk/|adj. 学术的，学院的
accelerate|/əkˈseləreɪt/|v. 加速，促进
accent|/ˈæksent/|n. 口音，重音
accept|/əkˈsept/|v. 接受，认可
access|/ˈækses/|n./v. 通道，进入
accident|/ˈæksɪdənt/|n. 事故，意外
accompany|/əˈkʌmpəni/|v. 陪伴，伴随
accomplish|/əˈkʌmplɪʃ/|v. 完成，实现
accord|/əˈkɔːrd/|n./v. 一致，符合
account|/əˈkaʊnt/|n./v. 账户，解释
accumulate|/əˈkjuːmjəleɪt/|v. 积累，积聚
```

> **说明**：你可以直接复制四六级/雅思/托福词表进去，建议先放 **500-1000词** 作为基础库。字段顺序：**单词|音标|释义**

### 1.2 保存到 iCloud Drive

- **电脑端**：直接保存到 `~/Library/Mobile Documents/com~apple~CloudDocs/`（即 iCloud Drive 文件夹）
- **手机端**：在"文件"App → iCloud Drive 中创建，用备忘录写好然后导出为 .txt

## 第二步：创建快捷指令

### 2.1 新建快捷指令

1. 打开 iPhone **快捷指令 App**
2. 右上角 `+` 新建快捷指令
3. 点击顶部名称，重命名为：**每日20词**

### 2.2 添加动作（按顺序）

#### 动作 1：获取文件

- 搜索并添加：**获取文件**
- 点击展开设置：
  - 关闭 `显示文稿选取器`（即让快捷指令自动找文件，不用每次手动选）
  - `文件路径` 填：`/vocabulary.txt`（如果在 iCloud Drive 根目录）
  - 或者勾选 `显示文稿选取器`（第一次运行时手动选一次，之后记住）

#### 动作 2：拆分文本

- 搜索并添加：**拆分文本**
- 设置：
  - `分隔符`：选择 `自定义`
  - 输入：`换行`（即每行拆分一个项目）

#### 动作 3：过滤空行（可选但建议）

- 搜索并添加：**过滤图像**（不用管它名字，实际过滤的是列表）
- 或者更简单：直接跳过，后面过滤

#### 动作 4：从列表中获取项目

- 搜索并添加：**从列表中获取项目**
- 设置：
  - `获取`：`随机项目`
  - `项目数量`：`20`
  - 关闭 `不重复`（或者开启，看你想要复习还是新词）

#### 动作 5：重复每个项目

- 搜索并添加：**重复每个项目**
- 内部添加以下动作：

  **a. 拆分文本（针对单词行内分隔）**
  - 搜索：**拆分文本**
  - 设置：
    - `输入`：`重复项目`（默认就是当前循环项）
    - `分隔符`：自定义，输入 `|`
  
  **b. 获取列表中的项目（分字段）**
  - 搜索：**从列表中获取项目**
  - 设置：
    - `输入`：`拆分文本的结果`
    - `获取`：`第一项`（这是单词本身）
  - 长按这个 `获取项目`，选择 `魔法变量`，改名为 `单词`
  
  - 再添加一个 **从列表中获取项目**：
    - `输入`：`拆分文本的结果`
    - `获取`：`第二项`（音标）
    - 改名为 `音标`
  
  - 再添加一个：
    - `输入`：`拆分文本的结果`
    - `获取`：`第三项`（释义）
    - 改名为 `释义`
  
  **c. 文本（格式化输出）**
  - 搜索：**文本**
  - 输入：
    ```
    🔤 单词：单词
    📢 音标：音标
    📖 释义：释义
    ---
    ```
    注意：点击 `单词`、`音标`、`释义` 字样，选择对应的魔法变量插入

#### 动作 6：合并文本

- 搜索：**合并文本**
- 设置：
  - `分隔符`：`新行`（或保留默认）

#### 动作 7：添加到备忘录

- 搜索：**创建备忘录**
- 设置：
  - `内容`：`合并文本的结果`
  - `文件夹`：选择一个文件夹（如"学习"）
  - `标题`：`2026-05-07 单词打卡`（可以用 `当前日期` 变量，搜索 `日期` 添加）

#### 动作 8：显示通知（可选）

- 搜索：**显示通知**
- 设置：
  - `标题`：`今日20词已生成`
  - `内容`：`已添加到备忘录，开始背诵吧！`

## 第三步：设置自动化

1. 在快捷指令 App 底部，切换到 **自动化** 标签
2. 点击 `创建个人自动化`
3. 选择触发条件：
   - **时间**：每天早晨 8:00（或你想背单词的时间）
   - 或者 **闹钟**：当你关闭起床闹钟时
4. 点击 `下一步`
5. 添加动作：**运行快捷指令** → 选择 **每日20词**
6. 点击 `下一步`
7. **关闭** `运行前询问`（否则每天早上要手动点一下）
8. 点击 `完成`

## 第四步：进阶玩法（推荐）

### 4.1 艾宾浩斯复习标记

在单词库中加一个字段标记复习状态：

```
abandon|/əˈbændən/|v. 放弃|2026-05-07|0
```
- 第四字段：加入日期
- 第五字段：复习次数

快捷指令过滤逻辑：优先抽复习日到的词，不够20个再补新词。

> 这需要更复杂的快捷指令逻辑（如果+循环），建议熟练基础版后再加。

### 4.2 语音朗读

在"重复每个项目"循环里，**文本动作之后**加一个：
- **朗读文本**：朗读 `单词` 变量
- 这样生成备忘录的同时，手机会自动念单词给你听

### 4.3 多词库切换

创建多个词库文件：
- `vocabulary_cet4.txt`
- `vocabulary_cet6.txt`
- `vocabulary_ielts.txt`

快捷指令开头加一个"选择列表"动作，让你选今天背哪个。

### 4.4 用 AI API 生成（最强方案）

如果你愿意用 API（DeepSeek/Kimi 很便宜），快捷指令可以：

1. **获取文件**（读取已学会的单词记录）
2. **获取剪贴板**（或输入你的学习目标）
3. **获取URL内容**（调用 DeepSeek API，Prompt：`根据我的已掌握词表，生成20个进阶单词，带音标和例句`）
4. **解析JSON** → 输出到备忘录

这需要 API Key 和网络请求知识，进阶时再搞。

## 常见问题

| 问题 | 解决 |
|------|------|
| 文件找不到 | 确认文件在 iCloud Drive，路径填写正确。首次建议开启"显示文稿选取器"手动选一次 |
| 分隔符不对 | 检查 `vocabulary.txt` 用的是英文 `|` 还是中文 `｜`，快捷指令里要一致 |
| 格式错乱 | 确保单词行内没有多余的 `\|` 符号，释义里不要带 `\|` |
| 重复单词 | 开启"从列表中获取项目"里的 `不重复` 选项 |
| 超过20个 | 检查"获取项目数量"是不是设成20 |
