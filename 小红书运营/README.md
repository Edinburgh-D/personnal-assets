# 小红书运营资料汇总

## 内容说明

本目录汇总了小红书运营相关的项目代码、采集工具和运营方案。

### 文件清单

| 文件/目录 | 说明 |
|:---|:---|
| `小红书闲鱼_三端自闭环运营方案_v1.0.md` | 三端（小红书+闲鱼+微信）自闭环运营方案 |
| `XHS_ALL_IN_ONE/` | 小红书一站式智能运营平台源码 |
| `需求采集脚本/` | 小红书需求采集 Python 脚本 |

### XHS_ALL_IN_ONE 项目

- **来源**: https://github.com/cv-cat/XHS_ALL_IN_ONE
- **功能**: 采集、分析、AI 创作、自动发布全链路闭环
- **技术栈**: Python (FastAPI) + React + SQLite

### 需求采集脚本

- **功能**: 基于关键词策略采集小红书用户需求笔记
- **输出**: JSON 格式需求分析报告（需配置 Cookie 后运行）

### 数据采集状态

- 数据库 `data/spider_xhs.db` 当前为空表结构，尚未写入实际采集数据
- 如需采集，需配置 `.env` 中的 Cookie 并运行采集脚本
