## 二、现有观赛渠道（竞品扫描）

### 官方/授权直播渠道
| 平台 | 类型 | 覆盖内容 | 社交功能 |
|------|------|----------|----------|
| **央视频APP** | 官方 | CCTV5直播/回放 | ❌ 无 |
| **腾讯体育** | 授权 | 部分赛事直播 | ⚠️ 弹幕弱 |
| **央视体育** | 官方 | CCTV全频道 | ❌ 无 |
| **虎牙直播** | 聚合 | 集锦/回放 | ✅ 弹幕强 |
| **BBC iPlayer** | 海外 | 英国地区 | ❌ 无 |
| **Discovery+** | 海外 | 欧洲地区 | ❌ 无 |

### 斯诺克内容社区
| 平台 | 类型 | 内容形态 | 问题 |
|------|------|----------|------|
| **抖音** | 短视频 | 赛事剪辑、教学 | 碎片化，无体系 |
| **B站** | 中长视频 | 解说、集锦 | 非专注斯诺克，分散 |
| **知乎** | 图文 | 赛事分析、问答 | 互动弱，无实时 |
| **贴吧/微博** | 论坛 | 话题讨论 | 杂乱，信息噪音大 |
| **虎牙斯诺克专区** | 直播 | 赛事直播+弹幕 | 但非独立App |

### 核心发现
- **没有一款"斯诺克专属"的综合App**（直播+数据+社区+商城）
- 现有渠道全是"观赛工具"，没有"球迷社区"
- 1.5亿观众看完比赛后，没有地方深入讨论、追踪数据、买装备

## 三、App可行性判断

### ✅ 可以做，而且时机很好

| 因素 | 判断 |
|------|------|
| **用户基数** | 1.5亿+潜在用户，足够支撑一个垂直App |
| **付费意愿** | 赵心童同款马甲卖断货，装备消费力强 |
| **赛事密度** | 世锦赛、英锦赛、大师赛、排名赛全年不断 |
| **版权可行性** | 央视有版权，可合作/导流，不自己做直播 |
| **差异化空间** | 现有渠道全是工具型，社区型完全空白 |
| **变现路径** | 装备电商（球杆/配件）、会员增值、广告 |

### ⚠️ 需要注意的风险

| 风险 | 说明 | 应对 |
|------|------|------|
| **赛事版权** | 直播版权在央视/腾讯，拿不到 | 不做直播，做"数据+社区+导流" |
| **受众偏男性** | 台球男性主导，女性比例在增长但仍是 minority | 做"轻社交"而非"社交App" |
| **赛事间歇期** | 大型赛事间隔几个月，用户留存挑战 | 做"训练/技术"内容填充淡季 |
| **竞品反应** | 虎牙/腾讯可能加码斯诺克社区 | 先发优势+垂直深度 |

## 四、建议的App形态

不做"另一个直播平台"，做**"斯诺克球迷的综合社区"**。

### 核心功能模块

| 模块 | 功能 | 对标 |
|------|------|------|
| **赛事中心** | 赛程、比分、排名、签表、历史数据 | FlashScore（纯数据） |
| **技术拆解** | AI辅助的击球路线分析、单杆回放 | 差异化功能 |
| **社区讨论** | 赛事话题、球员动态、技术交流 | 虎扑（垂直版） |
| **装备商城** | 球杆、台呢、皮头、配件 | 得物（装备版） |
| **个人中心** | 收藏球员、关注赛事、战绩统计 | 粉丝工具 |
| **训练辅助** | 击球视频分析、姿势纠正（AI） | 差异化功能 |

### 不做直播的原因
- 版权成本高、谈判复杂
- 央视/腾讯已有成熟渠道
- 我们的价值是"看完直播后"

### MVP建议
1. **Phase 1**：赛事数据（赛程/比分/排名）+ 社区话题
2. **Phase 2**：球员技术档案 + AI击球分析
3. **Phase 3**：装备推荐（跳转电商赚佣金）
4. **Phase 4**：训练视频上传 + AI姿势分析

## 五、与"台球圈子App"的关系

| 维度 | 台球圈子App | 斯诺克专属App |
|------|-------------|---------------|
| **受众** | 泛台球爱好者（中式/美式/斯诺克） | 斯诺克深度球迷 |

| **变现** | 门店抽佣、装备电商、SaaS | 装备电商、会员增值、广告 |
| **竞品** | B端SaaS空白 | 观赛工具空白 |
| **用户量** | 2.1亿泛台球 | 1.5亿+斯诺克观赛 |

**建议**：两个项目不冲突，甚至可以整合——"台球圈子"主打线下社交，"斯诺克中心"主打线上数据社区。未来可以账号互通。

## 六、关键结论

1. **热度无需验证**——1.5亿观众、15亿社媒曝光、31.7%央视市场份额，这是顶级体育赛事的数据
2. **市场空白明确**——没有斯诺克专属的综合社区App，现有全是观赛工具
3. **变现路径清晰**——装备电商（球杆复购率高）、会员数据服务、广告
4. **版权不是问题**——不做直播，做数据+社区，与央视互补而非竞争
5. **时机窗口**——赵心童/吴宜泽连续夺冠，热度正处于峰值，现在入局有话题红利
