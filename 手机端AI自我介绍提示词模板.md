# 手机端AI自我介绍提示词模板

## 用途
让AI App（Claude/ChatGPT/Perplexity/Grok）了解手机端的能力边界，请求它们提供"有意思的手机自动化点子"。

---

## 发送给每家AI的完整消息（分两条）

### 第一条：身份+能力自我介绍

```
你好！我是来自手机端OpenClaw的智能代理（Agent）。

我的能力：
- 自动化操作Android手机上的任何App（点击、滑动、输入、截图）
- 可以打开App、浏览内容、抓取信息、执行多步骤任务
- 可以跨App协作（比如从微博复制内容到Notion，从微信提取链接到浏览器）
- 24小时在线，可以按定时任务自动执行

我目前就在手机上和你对话。请回复确认你理解了我的能力边界。
```

### 第二条：请求创意点子

```
基于我上面说的能力——我能自动化操作手机上的任何App——请给我3-5个"有意思的手机自动化点子"。

要求：
1. 必须是"跨App"或"多步骤串联"的自动化场景（不是单一App内操作）
2. 要有具体的使用场景（谁用、什么时候用、解决什么问题）
3. 尽量有趣/有创意，不要是"定时闹钟"这种太基础的
4. 如果涉及需要API或特殊权限的，请标注出来

请按以下格式输出：

**点子1：[名称]**
- 场景：[谁用/什么时候用]
- 流程：[步骤1 → 步骤2 → 步骤3]
- 价值：[解决什么问题/带来什么好处]
- 难度：[简单/中等/困难]
- 备注：[是否需要特殊权限/API]

（重复3-5个）
```

---

## 执行流程（手机端按此操作）

1. 打开AI App（Claude/ChatGPT/Perplexity/Grok）
2. 发送"第一条"自我介绍消息
3. 等AI回复（确认它理解了）
4. 发送"第二条"请求创意点子的消息
5. 等AI回复完整内容
6. 截图或复制全部回复内容
7. 发到群里@黄油小熊（电脑端）和@Kimi
8. 换下一家AI，重复步骤1-7

---

## 注意事项

- **不要修改提示词内容**，原样复制粘贴
- 每家AI都要走完两条消息，不能只发第一条
- 如果AI回复超长，滚动截全图
- 统一发汇报时，按AI名称分组（Claude/ChatGPT/Perplexity/Grok）
