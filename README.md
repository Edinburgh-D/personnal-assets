## 🎯 核心资产

### 题库（v3.0 最终版）

**位置**：`题库/` 目录

**核心文件**：
- `父母指南_题库统一版_v3.0.json` — 开发直接接入的标准JSON
- `父母指南_完整题库收录_v3.0.md` — 全部场景+选项+解析，设计/文案参考

**24道题覆盖**：
- 0-3岁：超市打滚、乱跑不回家、二胎争宠、高敏感衣服、睡眠抗拒、发脾气打滚
- 4-6岁：分离焦虑、抢玩具、不肯吃饭、乱涂墙壁、怕黑、吵架绝交
- 7-12岁：考试藏卷、被孤立、偷拿零钱、沉迷游戏、抢遥控器、写作业磨蹭
- 13-18岁：锁门摔手机、深夜刷手机、拒绝沟通、撕卷子、恋爱、电竞退学

**4维度映射**：
| 选项 | 风格 | 画像 |
|------|------|------|
| A | 情绪回应型 | 温室园丁 🟢 |
| B | 规则锚定型 | 目标教练 🟡 |
| C | 探索追问型 | 侦探型父母 🔵 |
| D | 行动果断型 | 行动派指挥官 🟤 |
