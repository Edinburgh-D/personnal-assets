# 「父母指南」小程序 UI 设计图生成提示词

> 目标：为 8 个页面生成完整的设计稿，用于指导前端开发
> 风格：去 AI 化、有温度感、自然亲和、手绘温度
> 技术约束：适配 TailwindCSS、手机端 375px 基准宽度

---

## 一、项目概述

「父母指南」是一款微信小程序，帮助父母通过 28 道场景化题目了解自己的育儿风格。核心定位是"自我认知工具"，不是考试、不是评判。

**设计关键词**：温暖、安全感、不评判、陪伴感、自然
**去 AI 化关键词**：拒绝科技感、拒绝未来感、拒绝冷色调渐变、拒绝几何抽象、拒绝"智能"/"AI"等字样

## 二、全局设计系统

### 2.1 色彩系统

**基础色（所有页面共用）**：
| 用途 | 色值 | Tailwind 类名 |
|------|------|---------------|
| 背景色 | #F5F0EB | `bg-[#F5F0EB]` |
| 卡片白 | #FFFFFF | `bg-white` |
| 主文字 | #1A1A1A | `text-[#1A1A1A]` |
| 次文字 | #666666 | `text-[#666666]` |
| 辅助文字 | #999999 | `text-[#999999]` |
| 分隔线 | #E5E5E5 | `border-[#E5E5E5]` |
| 错误/警示 | #E74C3C | `text-red-500` |

**页面强调色（各页面不同，但都在暖色系内）**：
| 页面 | 强调色 | 色值 | 情绪定位 |
|------|--------|------|----------|
| 首页 | 暖橙 | #FF8C42 | 活力、邀请 |
| 信息收集页 | 暖杏 | #FFB366 | 温柔、关怀 |
| 答题页 | 暖米 | #F5E6D3 | 专注、沉浸 |
| 结果页 | 暖金 | #D4A574 | 成就、温暖 |
| 心路历程页 | 暖绿 | #8FBC8F | 成长、希望 |
| 海报分享页 | 暖橙 | #FF8C42 | 传播、能量 |
| 课程推荐页 | 暖蓝 | #6B8E8B | 信任、专业 |
| 历史记录页 | 暖紫 | #9B7EBD | 记忆、沉淀 |

**强调色使用规则**：
- 仅用于：主按钮背景、选中状态、关键图标、进度条填充
- 不得用于：大面积背景、文字主色、装饰性图形
- 透明度变体：`bg-opacity-10` 用于轻量背景点缀

### 2.2 字体系统

**去 AI 化字体策略**：不用系统默认无衬线，优先用有温度的字体
| 层级 | 字体 | 字号 | 字重 | 行高 | Tailwind 示例 |
|------|------|------|------|------|---------------|
| 品牌标题 | 圆润手写体（站酷快乐体/优设标题圆） | 28px | 700 | 1.3 | `text-[28px] font-bold` |
| 页面标题 | 系统圆体（PingFang SC / HarmonyOS Sans） | 22px | 600 | 1.4 | `text-[22px] font-semibold` |
| 卡片标题 | 同上 | 18px | 600 | 1.4 | `text-lg font-semibold` |
| 正文 | 系统无衬线 | 16px | 400 | 1.6 | `text-base leading-relaxed` |
| 辅助文字 | 同上 | 14px | 400 | 1.5 | `text-sm` |
| 标签/数据 | 同上 | 12px | 500 | 1.4 | `text-xs font-medium` |
| 场景描述 | 同上 | 16px | 400 | 1.8 | `text-base leading-loose` |

**去 AI 化要点**：
- 场景描述文字用宽松行高（1.8），模拟"慢慢说"的感觉
- 避免全大写、避免紧凑字间距
- 关键情绪词可以加粗但不换颜色

### 2.3 间距系统

基于 8px 网格：
| Token | 值 | Tailwind |
|-------|-----|----------|
| xs | 4px | `p-1` |
| sm | 8px | `p-2` |
| md | 16px | `p-4` |
| lg | 24px | `p-6` |
| xl | 32px | `p-8` |
| xxl | 48px | `p-12` |

**页面边距**：左右 20px（`px-5`）
**卡片内边距**：16px（`p-4`）
**元素间距**：12px（`gap-3`）
**区块间距**：24px（`mb-6`）

### 2.4 圆角系统

| 用途 | 值 | Tailwind |
|------|-----|----------|
| 小元素（按钮/标签） | 22px（半圆） | `rounded-[22px]` |
| 卡片 | 20px | `rounded-[20px]` |
| 输入框 | 12px | `rounded-xl` |
| 头像/小图标 | 50% | `rounded-full` |
| 弹窗 | 24px | `rounded-[24px]` |

### 2.5 阴影系统

不用大阴影，用"纸张叠加"感：
| 用途 | 值 | Tailwind |
|------|-----|----------|
| 卡片 | `0 2px 12px rgba(0,0,0,0.04)` | `shadow-sm` |
| 悬浮按钮 | `0 4px 16px rgba(255,140,66,0.3)` | 自定义 |
| 弹窗 | `0 8px 32px rgba(0,0,0,0.08)` | `shadow-lg` |

### 2.6 插画风格

**去 AI 化核心**：
- 用"手绘水彩/彩铅/蜡笔"风格，不用矢量扁平插画
- 人物比例不追求精准，可以有手绘的歪扭感
- 表情温暖、夸张、真实（不是 emoji，是有表情的小插图）
- 配色偏暖，和页面强调色同色系
- 背景留白，不堆砌装饰元素
- **绝对禁止**：几何抽象图案、科技感线条、渐变色块装饰、机器人/芯片元素

## 四、跨页面一致性规范

### 4.1 导航栏
- 所有子页面统一：左侧返回箭头（24px），中间页面标题（18px，600），右侧空或省略号菜单
- 背景白底，底部 1px 分隔线 #E5E5E5
- 高度 44px + 状态栏

### 4.2 按钮规范
| 类型 | 样式 | 状态 |
|------|------|------|
| 主按钮 | 强调色底 + 白字 + 圆角28 + 阴影 | 按下：缩小0.97 + 阴影减弱 |
| 次按钮 | 白底 + 强调色边框 + 强调色文字 | 按下：背景变浅 |
| 文字按钮 | 纯文字 + #999 | 按下：变#666 |
| 危险操作 | 纯文字 + #E74C3C | 按下：变暗红 |

### 4.3 空状态
- 插画：一只小动物（猫/狗/兔子）困惑地歪头
- 文案：温和，不责备。"还没有记录，开始你的第一次测试吧"
- 按钮：引导到测试入口

### 4.4 加载状态
- 不用 spinner，用"呼吸"动画：一个圆点放大缩小
- 文案："正在准备你的专属题目..."

### 4.5 错误状态
- 插画：一只小动物（猫）把东西打翻了
- 文案："哎呀，出了点小问题"
- 按钮："再试一次" + "返回首页"

## 五、去 AI 化检查清单

| 检查项 | 要求 | 禁止 |
|--------|------|------|
| 配色 | 暖色系为主，自然色 | 冷色调渐变、霓虹色、高饱和科技蓝 |
| 插画 | 手绘水彩/彩铅/蜡笔，人物有表情 | 矢量扁平、几何抽象、科技感线条 |
| 字体 | 圆润、人文、有温度 | 全大写、机械感、等宽字体 |
| 文案 | 口语化、有括号注释、有情绪 | "智能"/"AI"/"算法"/"精准"/"大数据" |
| 布局 | 宽松、留白、纸张感 | 紧凑、信息堆砌、仪表盘风格 |
| 动效 | 轻微、自然、弹性 | 闪烁、脉冲、机械滑动 |
| 图标 | 手绘风格、圆角 | 尖锐几何、线性图标、科技感 |
